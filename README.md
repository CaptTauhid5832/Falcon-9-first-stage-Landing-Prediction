# Falcon-9-first-stage-Landing-Prediction
Applied Data Science Capstone – IBM
🚀 Applied Data Science Capstone – IBM

This project is part of the IBM Applied Data Science Capstone. It focuses on analyzing SpaceX launch data to predict the success of Falcon 9 first-stage landings.

📌 Objectives
Collect data from APIs and datasets
Clean and preprocess the data
Perform exploratory data analysis (EDA)
Build models to predict landing success
🛠️ Tools & Technologies
Python
Pandas, NumPy
Matplotlib, Seaborn
Jupyter Notebook
📊 Key Outcome

The project identifies key factors that influence successful rocket landings, supporting data-driven decision-making.

📁 Dataset

The dataset includes SpaceX launch records, processed and cleaned for analysis.
